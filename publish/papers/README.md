# publish/papers/ — P2 paper channel

One folder per paper: `<paper-id>/` with `manuscript.tex`, `claims-cited.md`
(claim IDs + tiers at submission), `STATUS.md` (draft / internal-review /
submitted / in-revision / published), figures, version history. Paper prose is
manual-only (rule M1). At submission, tag `paper/<paper-id>/v<N>` freezes the
cited claim set (rule M2); no paper may cite a claim above its registered tier
(rule M3). First target: Packet A — "Reading-H vacuum selection in a corrected
Brazovskii-TECT model" (10-20 pages).
