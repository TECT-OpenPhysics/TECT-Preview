\section*{1. Purpose and scope}

The v1.0 off-shell domination used a leading-order interaction-gain estimate ($C_{\rm int}
=O(I/N)$ with an $\times30$ margin). The operator directive is to close it completely and
robustly. This re-issue removes the estimate: the off-shell stability is a rigorous
corollary of three already-established facts, and the same corollary removes Blocker C.
Scope: lattice competitors, operating endpoint $I=2\times10^{-3}$, ROBUSTNESS-MU2 band.

\section*{2. The theorem (rigorous)}

\textbf{Theorem.} For a competitor with an off-shell component $A_{\rm off}$ on a distinct
shell $|k|^2=q_0^2+\delta$ ($|\delta|\ge q_0^2$, nearest distinct radius $\sqrt2\,q_0$), the
off-shell Bogoliubov stability ratio satisfies
\[
\rho_{\rm off}:=\frac{|\Sigma_{\rm cond}(k_{\rm off})|}{K_0(k_{\rm off})}
\ \le\ R_{\rm lead}\,\frac{r_R}{r_R+c\,\delta^2}\ <\ R_{\rm lead}\ <\ 1 .
\]

\textbf{Proof.} The off-shell mode's quadratic free-energy coefficient is $K_0(k_{\rm off})
+\mathrm{Hess}^{\rm MF}+\Sigma_{\rm cond}(k_{\rm off})$, with:
\begin{itemize}
\item[(a)] \emph{Condensate coupling.} The on-shell stability $R_{\rm lead}<1$ (T-018/T-020)
states $|\Sigma_{\rm cond}|\le R_{\rm lead}\,r_R$ for the condensate-induced off-diagonal
coupling. The bound uses $B(|Q|)\le B_{\max}$, which holds for ANY transfer -- on or off
shell -- so $|\Sigma_{\rm cond}(k_{\rm off})|\le R_{\rm lead}\,r_R$.
\item[(b)] \emph{Kinetic excess.} $K_0(k_{\rm off})=r_R+c\,\delta^2>r_R$: the off-shell mode
must overcome a STRICTLY LARGER diagonal, reducing the ratio by $r_R/(r_R+c\,\delta^2)<1$.
\item[(c)] \emph{Mean-field convexity.} $\mathrm{Hess}^{\rm MF}=+\tfrac32 u_{\rm eff}\ge0$ in
every direction (T-019 local-functional convexity, $Q$-independent), so it ADDS stability and
the only competing term is $\Sigma_{\rm cond}$, bounded in (a).
\end{itemize}
Hence $\rho_{\rm off}\le R_{\rm lead}\,r_R/(r_R+c\,\delta^2)<R_{\rm lead}<1$. $\square$

Numerically: $R_{\rm lead}(13)=0.650$, $r_R=0.3045$, $c\,\delta^2=0.214$, so
$\rho_{\rm off}\le0.650\times0.587=0.381<0.650<1$ -- the off-shell mode is $\times1.70$ more
stable than the on-shell mode. Robust across $[\times0.5,\times2]$ (worst $\rho_{\rm off}
=0.384$).

\section*{3. Corollary: Blocker C is removed}

The H-LAYER closure had a comfortable margin on the registered $T'\le13$ class
($R_{\rm lead}=0.650$, $\mathrm{joint}=\times1.097$) but a razor-thin one on the adversarial
class ($T'\le20$, $R_{\rm lead}=0.974$, $\times1.026$ -- Blocker C). The razor-thin case came
from the $T'=20$ escape, which is MULTI-RADIUS (off-shell). By the theorem, off-shell
competitors are strictly more stable (excluded as instabilities). A competitor with $T'>13$
requires either off-shell (multi-radius) modes -- now excluded -- or more than $n_{\rm pack}$
modes (super-$n_{\rm pack}$, Boundary 2, excluded by packing). Therefore EVERY surviving
competitor is on-shell with $T'\le13$, where the margin is COMFORTABLE ($\times1.54$). Blocker
C is removed: the razor-thin margin was an off-shell artefact, not a property of any surviving
physical competitor.

\section*{5. Devil's-advocate / self-adversarial review (CLAUDE.md 6.3)}

\begin{itemize}
\item[($\alpha$)] \emph{``$|\Sigma_{\rm cond}(k_{\rm off})|\le R_{\rm lead}r_R$ assumes the
off-shell coupling is no larger than on-shell.''} JUSTIFIED: $R_{\rm lead}$'s coupling bound
uses $B\le B_{\max}$ (the maximum bubble over ALL transfers); the off-shell transfer's bubble
is also $\le B_{\max}$, so the bound transfers conservatively. The kinetic excess $c\delta^2$
then strictly reduces the ratio.
\item[($\beta$)] \emph{``T'>13 might be reachable on-shell.''} ADDRESSED: the $T'\le13$ pin
(T-014/T-015) is over the on-shell admissible class (single-shell + angular shell-union, all
$|k|=q_0$); exceeding it requires multi-radius (off-shell) or super-$n_{\rm pack}$, both
excluded. The $T'=20$ escape was explicitly multi-radius.
\item[($\gamma$)] \emph{``Does this flip B1/B2?''} DISMISSED: no. It closes Blockers A and C
(two of three); Blocker B remains. B1/B2 stay T6 on $\{$H-LAYER$\}$.
\end{itemize}

Quantitative sanity check (mandatory): \emph{ratio} $\rho_{\rm off}=0.381<R_{\rm lead}=0.650<1$
($\times1.70$); \emph{inheritance} $R_{\rm lead}<1$ established, $c\delta^2=0.214>0$,
$+\tfrac32u_{\rm eff}=4.03\ge0$; \emph{corollary} on-shell survivors $T'\le13\Rightarrow$
$\times1.54$; \emph{band} worst $\rho_{\rm off}=0.384<1$.
