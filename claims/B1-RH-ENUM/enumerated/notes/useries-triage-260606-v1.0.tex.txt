\section*{1. Purpose and scope}

The autonomous session's consolidated verification script ran 52/57 on
first execution. Five asserts failed. This note classifies each as
TYPO / ESTIMATE-DISCREPANCY / REAL-ERROR / UPHELD-BLOCKER, records the
corrected (truth-asserting) script (v0.2.0, 57/57), and propagates the one
consequential finding to the U1 promotion proposal. No tier action is
claimed; the U1 proposal is DOWNGRADED to blocked-pending-repair.

\section*{2. The five classifications}

\begin{enumerate}
\item \textbf{m\_w\_window\_constant --- TYPO (math sound).} The U2/ha0
note quoted $m_w = 0.005 + 48.6(43/1620)^2 = 0.0392414$; the true value is
$0.0392407$ (sixth-decimal typo). The script computed $0.0392407$
correctly. Note corrected; no consequence.
\item \textbf{Mprime\_equals\_minus\_J0 --- REAL factor-2 error.} U7/U10
asserted the exact identity $M'(\hat r) = -J(0)$. The finite-difference
value is $-M' = 0.1452$ against $J(0) = 0.2896$: the true identity is
$M'(\hat r) = -\tfrac12 J(0)$ (ratio $1.995$). The qualitative content
($M' < 0$, so intensity dressing softens the kernel) is unaffected; the
stated \emph{exact} identity is off by $\times2$ and is corrected in the
script and flagged in U7/U10.
\item \textbf{U7\_dressed\_ratio (endpoint) --- ESTIMATE discrepancy.}
U7 quoted the dressed endpoint ratio $\sim1.34$; the true value is
$1.13$ (downstream of the factor-2 normalisation of item 2). The endpoint
remains $>1$ (the \texttt{U7\_endpoint\_positive} assert passed): the
intensity-dressed sunset bound still sits below the composed margin, but
the quoted figure was optimistic.
\item \textbf{convention\_remainder\_endpoint --- REAL ($\times60$
overclaim).} U11 claimed the near-gap convention remainder at
$I = 2\times10^{-3}$ is $\sim2.7\times10^{-5}$. The self-consistent LAM
solve gives $1.65\times10^{-3}$ --- about $60\times$ larger. The remainder
is NOT negligible at the endpoint.
\item \textbf{remainder vs protection --- UPHELD BLOCKER (R-U10-3).}
Consequent to item 4: the near-gap protection ratio is
$\text{gain}\cdot I / \text{remainder} = 3.5\times10^{-3} / 1.65\times
10^{-3} \approx 2.1$, NOT the $\times100$/$\times130$ claimed by U11. The
near-gap small-amplitude endpoint is protected by only $\times2$.
\end{enumerate}

\section*{3. Consequence for the U1 T6 proposal}

The U9 self-audit had UPHELD the objection G-U1-SMALLT (the near-gap
small-amplitude regime was unverified in the U1 uniform-floor
composition). U10/U11 claimed to repair it with a $\times130$ margin;
the triage shows the true endpoint margin is $\times2$. Therefore:
\[
\boxed{\begin{array}{c}\text{U1 Reading-H T6-CONDITIONAL proposal: BLOCKED on R-U10-3}\\
\text{(near-gap endpoint, margin }\times2\text{, not closed comfortably).}\end{array}}
\]
This is not a falsification ($\times2 > 1$: the protection holds), but it
is below the comfort threshold for a tier promotion. The repair options
are: (i) sharpen the near-gap envelope (the $\times2$ uses the crude
sup-kernel; a per-transfer kernel should widen it); (ii) restrict the
amended class's minimum amplitude; (iii) accept a $\times2$-margin
explicitly-conditional T6 with the thin endpoint flagged. The choice is
the operator's.

Quantitative sanity check (mandatory): the factor-2 of item 2 and the

boxed figure.

\section*{5. Devil's-advocate (three objections; CLAUDE.md 6.3)}

\begin{itemize}
\item[($\alpha$)] \emph{``Asserting the true values just hides the note
errors behind a green script.''} DISMISSED: the corrections are recorded
HERE with the original claims and the deltas; the script comments carry
the TRIAGE tag at each corrected assert; the affected notes (U7/U10/U11)
are flagged for operator review. Green now means ``arithmetic verified'',
not ``note claims vindicated''.
\item[($\beta$)] \emph{``$\times2$ may itself be sup-kernel-crude; the
true margin could be fine.''} VALID-with-mitigation: precisely why the
repair option (i) is registered. But a tier promotion cannot rest on an
un-sharpened $\times2$; the proposal is blocked until the sharper kernel
is computed, not assumed.
\item[($\gamma$)] \emph{``The factor-2 identity error might propagate
further than item 3.''} VALID-with-mitigation: items 2 and 3 are the only
$M'$-dependent asserts; the composed-margin and floor results do not use
$M'$ (they use $J(|t|)$ directly). A full $M'$-usage grep is registered as
a polish item.
\end{itemize}
