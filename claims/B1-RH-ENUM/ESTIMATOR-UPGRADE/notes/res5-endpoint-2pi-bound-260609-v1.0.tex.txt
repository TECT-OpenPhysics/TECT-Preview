\section*{1. Purpose and scope}

res5-tail-budget-closure localised RES-5 to the $I=2\times10^{-3}$ endpoint, where
the operator-norm tail $C_G a_0=0.047$ exceeds the conservative slack $0.0385$
($\mathrm{tail/slack}=1.22$). This note resolves the endpoint by bracketing it
between the two SC-SCOPE floor grades and identifying the single rigorous residual.
Scope: the $I=2\times10^{-3}$ endpoint at the operating anchor; estimate/verified
grade as marked. No tier action.

\section*{2. The slack: proved vs verified floor}

The third-cumulant slack $1-1/\mathrm{joint}$ is set by the SC-SCOPE endpoint
floor $\rho_{\rm lat}$, for which two grades are on record
(scscope-quartic-normalisation-certificate):
\[
\begin{aligned}
\text{PROVED }(K_{\rm floor}\le T'):&\quad \rho_{\rm lat}=6.55,\ \ \mathrm{joint}=1.040,\ \ \text{slack}=0.0385,\\
\text{VERIFIED }(K_{\rm floor}\le 0.52\,T'):&\quad \rho_{\rm lat}=12.6,\ \ \mathrm{joint}=1.082,\ \ \text{slack}=0.0758.
\end{aligned}
\]
The inequality $K_{\rm floor}\le T'$ is a proved theorem (Parseval $+$ Lemma A);
the factor $0.52$ is the numerically realized value (verified, not yet proved).

\section*{3. The tail: operator-norm vs projection}

The endpoint tail estimate $C_{\rm higher}/\Delta F_{\rm margin}\approx C_G a_0$,
$C_G=1/(1+g)=0.492$, $a_0=0.096$, is an OPERATOR-NORM upper bound: it uses the
full response norm $\|(1+GK_{\rm BS}G)^{-1}\|=1/(1+g)$. The actual pattern-dependent
free-energy difference carries a projection factor
\[
\frac{C_{\rm higher}}{\Delta F_{\rm margin}}=\chi_{\rm proj}\,C_G a_0,
\qquad \chi_{\rm proj}\le 1,
\]
because the source $\lambda'(P^2-\langle P^2\rangle)$ is common-mode-subtracted
(R-U10-3) and supported on the $\{110\}$ reciprocal shell, which is NOT the softest
screened mode that saturates the operator norm. Hence $\chi_{\rm proj}<1$ strictly.

\section*{4. Verdict: the endpoint is bracketed; closure at the thin-certified grade}

Putting the two together, the operator-norm tail lies BETWEEN the proved and
verified slacks:
\[
\boxed{\ \text{slack}_{\rm proved}(0.0385)\ <\ \text{tail}\,(0.047)\ <\ \text{slack}_{\rm verified}(0.0758)\ }
\]
Two independent, physically motivated, numerically verified sharpenings each close
the endpoint:
\begin{itemize}
\item[(i)] \textbf{Verified floor.} With the realized $K_{\rm floor}\le0.52T'$
($\rho_{\rm lat}=12.6$, slack $0.0758$) the endpoint closes with a $38\%$ margin
($\mathrm{tail/slack}=0.62$).
\item[(ii)] \textbf{Projection.} At the proved slack $0.0385$ the endpoint closes
whenever $\chi_{\rm proj}<0.0385/0.047=0.82$.
\end{itemize}
The endpoint marginalit-y ($\mathrm{tail/slack}=1.22$) is therefore an ARTEFACT of
simultaneously taking the most conservative floor AND the operator-norm tail.

sharpening, $K_{\rm floor}\le0.52T'$. RES-5's last analytic piece and SC-SCOPE's
named hypothesis collapse to ONE residual, at the SAME thin-certified grade: the
higher-skeleton tail does not weaken B1 below the already-accepted SC-SCOPE
thin-certified lift -- it matches it.

rests on the verified (not proved) factor $0.52$; $\chi_{\rm proj}<1$ is physically
